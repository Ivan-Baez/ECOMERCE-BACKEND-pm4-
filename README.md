## Deployment
El backend está desplegado en Render y conectado a Neon PostgreSQL.  
URL: https://pm4-backend-5foa.onrender.com/api  
Base de datos: Neon (PostgreSQL con conexión SSL). 
## 🚀 Features

- Autenticación con JWT
- CRUD de productos
- Gestión de usuarios
- Roles de administrador y usuario
- Documentación con Swagger
- Base de datos PostgreSQL
- Deploy en Render
- ## 🛠️ Tecnologías

- NestJS
- TypeScript
- PostgreSQL
- TypeORM
- Swagger
- JWT
- Render
- Neon
